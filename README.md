登录页面账号：admin，密码：123456，写死的

## 功能

1. 学生增删改查
2. 教师增删改查（教师分行政人员、专任教师）
3. 根据姓名、专业、系部对学生、教师进行查询
4. 导出、导入学生、教师数据
